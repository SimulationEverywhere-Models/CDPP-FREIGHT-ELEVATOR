Each atomic and coupled model is inside one folder, the atomic models are: Evehicle, Econtrol and Ecall. Tha coupled models are Elevator (Econtrol + Evehicle) and Freight Elevator (Ecall + Elevator).

For each of the models the following files are included: .h, .cpp, .ma, .ev, .log, .bat.

The simulation (once each model had been compiled) can be run from the batch file included.
